# Security Policy

## Supported Versions

現在サポートされているバージョン:

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅ |

## Reporting a Vulnerability

セキュリティ上の脆弱性を発見した場合は、**公開 Issue ではなく** プライベートな方法でご報告ください。

### 報告方法

GitHub の [Private vulnerability reporting](https://github.com/<owner>/lyra/security/advisories/new) を使用してください。

報告には以下を含めてください:

- 脆弱性の種類と影響範囲
- 再現手順
- 影響を受けるバージョン
- 可能であれば、修正案や緩和策

### 対応タイムライン

- **確認**: 3 営業日以内に受領確認をお送りします
- **評価**: 7 営業日以内に調査結果をお伝えします
- **修正**: 重大度に応じて対応します

ご報告いただいた内容は、修正リリースまで非公開として扱います。
